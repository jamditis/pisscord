# Pisscord

A private, peer-to-peer Discord clone for trusted groups.

There are two roles here:
1.  **The Builder (YOU)**: You need to turn this code into a program.
2.  **The User (YOUR FRIEND)**: They just run the program you give them.

---

## ðŸ› ï¸ Instructions for YOU (The Builder)

You need to do this **once** on your computer.

1.  **Cleanup**: If you have a folder named `dist` in this directory, **delete it**.
2.  **Install Node.js**: Download and install the "LTS" version from [nodejs.org](https://nodejs.org/).
3.  **Open Terminal**: Open Command Prompt or PowerShell in this unzipped folder.
4.  **Install Tools**: Type this and hit enter:
    ```bash
    npm install
    ```
5.  **Create the App**: Type this and hit enter:
    ```bash
    npm run dist
    ```
    *(This takes about 1-2 minutes)*

6.  **Find the App**:
    *   Go to the `dist` folder.
    *   Look for a file named `Pisscord Setup 1.0.0.exe`.

## ðŸ“¦ Instructions for YOUR FRIEND

1.  **Send the File**: Email or send the `Pisscord Setup 1.0.0.exe` file to your friend.
2.  **Run**: Tell them to double-click it. It will install the app to their computer and open it.
    *   *Note: Since you are a private developer, Windows might say "Unknown Publisher". Tell them to click "More Info" -> "Run Anyway".*
3.  **Connect**:
    *   Open the app yourself.
    *   Go to "Voice Lounge".
    *   Copy your ID and send it to them.
    *   They paste it and click Connect.
