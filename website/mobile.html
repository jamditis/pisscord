Installation

Get Pisscord on your Android device. Currently available as a direct APK download - iOS support coming soon.

Download APK

Requires Android 8.0 or later

1

Download the APK

Go to the GitHub releases page and download the latest app-debug.apk file to your Android device. You can do this directly on your phone's browser.

File will be saved to your Downloads folder
2

Enable Unknown Sources

Android blocks apps from unknown sources by default. When you try to install, you'll be prompted to allow installation from your browser or file manager.

Go to Settings → Apps → Special access → Install unknown apps
3

Install & Launch

Tap the downloaded APK file to install. Once installed, open Pisscord from your app drawer. Grant camera and microphone permissions when prompted.

Manual Updates

The mobile app doesn't auto-update yet. Check GitHub releases periodically for new versions, or follow us for announcements.

Troubleshooting

Common mobile issues and solutions.

App won't install

Make sure you've enabled "Install unknown apps" for your browser or file manager. Go to Settings → Apps → Special access → Install unknown apps and enable it for the app you're installing from.

No audio in calls

Check that you've granted microphone permissions. Go to Settings → Apps → Pisscord → Permissions and enable Microphone. Also check your phone's volume and that you're not on silent mode.

Camera not working

Ensure camera permissions are granted. Some phones also require storage permissions for camera functionality. Check Settings → Apps → Pisscord → Permissions.

Connection keeps dropping

P2P connections can be affected by mobile network quality. Try switching to WiFi if on cellular data. Some corporate or school WiFi networks block WebRTC traffic.

App crashes on launch

Try clearing app data: Settings → Apps → Pisscord → Storage → Clear Data. If issues persist, uninstall and reinstall the latest APK version.

Battery drain

Voice and video calls use significant battery. This is normal for real-time communication apps. Disable video when not needed to conserve battery. The app uses standard WebView and doesn't run background services.

Found a bug?

Report issues in the #issues channel in the app, or open an issue on GitHub. Include your Android version and device model.
